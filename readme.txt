My Social Share Buttons Plugin will help you to add Facebook Like, Twitter, Google+, Buffer, LinkedIn, StumbleUpon, Pinterest Share Buttons after post.

== Description ==
Use My Social Share Buttons Plugin to add Facebook Like, Twitter, Google+, Buffer, LinkedIn, StumbleUpon, Pinterest Share Buttons after post contents.

<strong>Features:<strong>

* Smart Pinterest share, allows you to set the image or get the first image automatically

* Stylish Buffer button show.

* All buttons can show horizontal counts.

* Show buttons after post contents by default after activate this plugin.

Support Blog: http://ataulswebdesigns.com/social-share-buttons-plugin/

 
== Installation ==

1. Upload `my-social-share-buttons.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! You are done!

Now you will be able to see this share buttons bellow every single post.
